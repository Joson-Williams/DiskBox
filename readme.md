# DiskBox 项目
DiskBox是一个开源、自由的磁盘分区和数据恢复软件，设计之初是为了代替DiskGenius等软件。
</br></br>
DiskBox基于GNU GPL v3协议开源，它是由GNU发布的：[GNU GENERAL PUBLIC LICENSE](https://www.gnu.org/licenses/gpl-3.0.txt)。</br></br>
